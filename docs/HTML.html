W3C is an organization that works full time to better web standards for everyone in the world. This organization is well recognized by most of the big time tech companies. Since the start of the organization they have had one main goal, Provide the access of web to everyone. This was to be done regardless of their monetary and basic resources they lack. Hypertext Markup Language, HTML is the most commonly used language to create websites. When any search is done to access a webpage the DNS and HTTP return HTML pages. Cascading Style Sheets, CSS are used to customize the HTML web pages. Customizing using CSS can be done in many different ways. Inline, styling and customizing per each line in the code. Internal, customizing each element at the top of the HTML code. Finally External, in this method you set instructions for each element to execute outside the file and link it at the top of the HTML.
