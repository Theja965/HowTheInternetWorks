IP Addresses, Packets and Routing

As mentioned before each PC has its own unique IP address. IP addresses are similar to mailing addresses. A mailing address includes house number, street, city, zip code and country. Similarly an IP address is a 32 bit number that contains information like country, region subnetwork and so on. Country and region are in the beginning parts of an IP address. Subnetwork and device number are in the later parts of the address. IP addresses make it possible to retrieve and send information to and from other locations. As addressed in the DNS article, it is essentially a GPS. If the DNS is a GPS, the packets of information moving is a car directed by the GPS.
